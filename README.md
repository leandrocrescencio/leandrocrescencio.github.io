## CV-Resume 
 Using the Pillar Resume Template as a Jekyll site for Github Pages

*  Fully Responsive
*  HTML5 + CSS3
*  Built on Bootstrap 4
*  **SCSS** source files included
*  6 Colour Schemes
*  1000+ FontAwesome 5 icons
*  Compatible with all modern browsers
